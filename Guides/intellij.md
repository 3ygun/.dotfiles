## Commands

Command | Discription
--- | ---
`⌘`+`F12` | See method names in current class
`⌘`+`⇧`+`N` | Search for any file by name
`Ctrl`+`⇧`+`F` | Search for a text string
`⌥`+`⇧`+`⌘`+`L` | Reformate file
`⌥`+`⌘`+(`←` or `→`) | Move forward or backward in "jump to" movements
