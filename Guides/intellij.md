## Commands

Command | Discription
--- | ---
`Command`+`F12` | See method names in current class
`Command`+`Shift`+`N` | Search for any file by name
`Ctrl`+`Shift`+`F` | Search for a text string
`⌥`+`⇧`+`⌘`+`L` | Reformate file
