# Install

Add a `DOTFILES` environment variable

## Bash Instructions

1. Clone to `~/`
2. Run `cp ~/.dotfiles/.bash_profile ~/.bash_profile`
3. Run `cp ~/.dotfiles/.bashrc ~/.bashrc`
4. Reload terminal
